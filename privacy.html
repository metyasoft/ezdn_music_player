EZDN Player

EZDN Player는 사용자의 개인정보를 소중하게 생각하며, 개인정보 보호를 최우선으로 합니다. 본 앱은 사용자의 개인정보를 수집, 저장, 전송하지 않습니다.

1. 개인정보의 처리 목적

EZDN Player는 다음과 같은 이유로 개인정보를 처리하지 않습니다:

  • 본 앱은 순수하게 로컬 음악 재생을 위한 플레이어입니다
  • 사용자 계정 시스템이 없습니다
  • 서버와의 데이터 통신이 없습니다
  • 외부 광고 또는 분석 도구를 사용하지 않습니다

2. 수집하는 정보

EZDN Player는 어떠한 개인정보도 수집하지 않습니다.

2.1 로컬 데이터 접근

앱은 다음의 로컬 데이터에만 접근하며, 이는 기기 내에서만 처리됩니다:

  • 다운로드 폴더: 사용자가 다운로드한 MP3 파일에 접근하여 재생 목록에 추가
  • 클립보드: 사용자가 복사한 이미지를 앨범 아트로 적용하기 위해 클립보드 접근 (사용자가 명시적으로 롱프레스 후 붙여넣기를 선택한 경우에만)
  • 음악 파일: 재생을 위해 기기 내 MP3 파일 접근

중요: 위의 모든 데이터는 사용자 기기 내에서만 처리되며, 외부 서버로 전송되거나 저장되지 않습니다.

3. 개인정보의 처리 및 보유 기간

EZDN Player는 개인정보를 수집하지 않으므로, 보유하거나 처리하는 개인정보가 없습니다.

사용자가 앱 내에서 설정한 모든 정보(재생 목록, 앨범 아트, 메타데이터 수정 등)는 사용자의 기기에만 저장되며, 앱을 삭제하면 함께 삭제됩니다.

4. 개인정보의 제3자 제공

  • 앨범 아트 이미지 검색: 사용자가 선택한 경우 검색 엔진으로 이동하여 앨범 아트 이미지를 검색할 수 있습니다
