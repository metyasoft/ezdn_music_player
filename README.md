# EZDN Player 지원 페이지

App Store Connect용 지원 URL 웹사이트입니다.

## 📁 포함된 파일

- `index.html` - 메인 지원 페이지
- `privacy.html` - 개인정보처리방침
- `EZDN_Player_01.png` - 스크린샷 1
- `EZDN_Player_02.png` - 스크린샷 2
- `EZDN_Player_03.png` - 스크린샷 3

## 🚀 GitHub Pages로 배포하기 (5분 완성)

### 1단계: GitHub 저장소 만들기
1. GitHub에 로그인
2. 새 저장소(repository) 생성
3. 저장소 이름: `ezdn-player-support` (원하는 이름)
4. Public으로 설정
5. README 체크 해제
6. Create repository 클릭

### 2단계: 파일 업로드
1. 생성된 저장소 페이지에서 "uploading an existing file" 클릭
2. 이 폴더의 모든 파일을 드래그 앤 드롭
   - index.html
   - privacy.html
   - EZDN_Player_01.png
   - EZDN_Player_02.png
   - EZDN_Player_03.png
3. "Commit changes" 클릭

### 3단계: GitHub Pages 활성화
1. 저장소의 Settings 탭 클릭
2. 왼쪽 메뉴에서 "Pages" 클릭
3. Source에서 "main" 브랜치 선택
4. Save 클릭
5. 몇 분 후 URL 생성: `https://[당신의username].github.io/ezdn-player-support`

### 4단계: App Store Connect에 입력
생성된 URL을 App Store Connect의 "지원 URL"에 입력하세요!

## 💡 대안 방법들

### Notion 사용하기
1. Notion에서 새 페이지 생성
2. index.html의 내용을 복사해서 Notion에 작성
3. 이미지들도 업로드
4. 우측 상단 "공유" → "웹에서 공유" 활성화
5. 생성된 링크 사용

### Google Sites 사용하기
1. sites.google.com 접속
2. 새 사이트 만들기
3. 텍스트와 이미지 드래그 앤 드롭으로 배치
4. 게시 → URL 복사
